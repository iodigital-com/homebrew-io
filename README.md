# iO Homebrew Tap

[Homebrew](https://brew.sh) tap for iO tools.

## Install

```bash
brew tap iodigital-com/io
brew trust --tap iodigital-com/io
```

## Update

```bash
brew update
```

## Uninstall

```bash
brew untap iodigital-com/io
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `claudio` | CLI for switching between Claude Code projects with different API keys |
